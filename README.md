По сути на протяжении недели и формировалось приложение под реальную задачу:
ассистент по работе над проектом в корпоративном гитлабе с локальной нейронкой.

Он уже может проводить ревью, работать с доской задач и работать с файловой системой
запускаясь либо в рамках пайплайна в корпоративном гитлабе либо как консольное приложение с чатом.
